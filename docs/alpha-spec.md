# Alpha Specification

## Purpose

Alpha validates the core messaging loop using professional architecture and calm UX.

---

## Included in Alpha

- Session-based authentication
- One-to-one direct messages
- Plain text messaging
- WebSocket realtime transport
- Message delivery states
- Presence and typing indicators
- Motion and visual foundations

---

## Out of scope

- Group conversations
- Media messages
- Message editing or deletion
- End-to-end encryption
- Push notifications
- Search
- Voice or video

---

## Definition of Done

Two users can reliably exchange real-time messages with correct delivery states and smooth UX, while all CI gates pass and documentation is complete.
